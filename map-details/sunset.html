SUNSET

// LOCATION
Los Angeles, USA

// OVERVIEW

A map with a classic three-lane, two-site layout set in a vibrant Los Angeles neighborhood. Sunset heavily emphasizes mid-control, with the team that dominates the central area gaining a significant strategic advantage for attacking either site.

// UNIQUE FEATURE

Mid Control Focus: Similar to Pearl, Sunset has no major movement gimmicks. Its core design revolves around the battle for Mid Courtyard. Winning mid control allows for splits onto both sites and cuts off defender rotations.
