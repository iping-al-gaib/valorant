ICEBOX

// LOCATION
Bennett Island, Russia

// OVERVIEW

An arctic excavation site, Icebox is a complex battlefield with significant verticality. Both bomb sites offer multiple angles of attack and elevated positions, making aim and awareness critical for both teams.

// UNIQUE FEATURE

High Verticality & Ziplines: Both sites feature many crates and structures to climb on, creating multi-level firefights. The A site also has horizontal ziplines that allow for rapid movement across elevated areas.
