ABYSS

// LOCATION
Secret base of the Hourglass Organization

// OVERVIEW

A two-site map set on the edge of the world, Abyss features treacherous cliffside paths. Control of the narrow bridges is key, as one wrong step can lead to a fatal fall...

// UNIQUE FEATURE

Fall Hazards: The map has no outer boundaries, allowing players to be knocked off the edge for an environmental kill.
