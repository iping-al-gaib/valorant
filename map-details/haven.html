HAVEN

// LOCATION
Thimphu, Bhutan

// OVERVIEW

Set in an ancient monastery, Haven is the only map in the competitive pool with three bomb sites (A, B, C). This unique layout stretches defenses thin and places a heavy emphasis on map control, rotations, and information gathering.

// UNIQUE FEATURE

Three Bomb Sites: The presence of A, B, and C sites forces defenders to cover more ground and makes retakes more complex. Attackers have more options for their final execute, making the map strategically diverse.
